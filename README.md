# pactarno
Pactarno is a NodeJS boilerplate built top on ExpressJS with Typescript support

### Technology Stacks
- NodeJS
- Typescript
- ExpressJS
- [Prisma](https://www.prisma.io/docs) (ORM)
### Installation
- Install the dependencies
  ```
   yarn install
  ````
  or if you're using NPM
  ```
  npm install --save
  ```
